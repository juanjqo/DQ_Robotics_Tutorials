## DQ Robotics (C++)

### Install pre-requisites

```shell
sudo apt install build-essential git cmake g++ libeigen3-dev fakeroot
```
### Clone the repository

```shell
git clone https://github.com/dqrobotics/cpp.git
```

### Build and install

```shell
cd cpp
mkdir build && cd build
chmod +x debian/rules
fakeroot debian/rules clean
fakeroot debian/rules build
fakeroot debian/rules binary
```

```shell
cd ..
sudo dpkg -i .libdqrobotics*.deb
```

### Uninstall

```shell
cd cpp/build
sudo xargs rm < install_manifest.txt
```

<img src="/img/computer_cpp.svg" alt="drawing" width="300"/>

<div>&nbsp;&nbsp;</div>

### Install pre-requisites

```shell
brew install eigen
```

### Build and install

```shell
cd cpp
mkdir build && cd build
cmake ..
make
sudo make install
```

### Clone the repository

```shell
git clone https://github.com/dqrobotics/cpp.git
```

### Uninstall

```shell
cd cpp/build
sudo xargs rm < install_manifest.txt
```

```shell

.\bootstrap-vcpkg.bat

cmd /c mklink /d c:\Tools\vcpkg c:\vcpkg
cd ~
```
